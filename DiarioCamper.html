🔥 Configurazione Firebase (sincronizzazione famiglia)

Firebase è un database gratuito di Google. Una volta configurato, tutti i tuoi dispositivi e la famiglia vedono gli stessi dati in tempo reale.

📖 Istruzioni passo-passo (5 minuti)
  1. Vai su console.firebase.google.com (accedi con Google)
  2. Clicca "Aggiungi progetto" → nome: diario-camper → continua fino a fine
  3. Nel menu a sinistra: Build → Realtime Database → Create Database
  4. Scegli Start in test mode → Abilita
  5. Nel menu a sinistra: Impostazioni progetto (icona ⚙️) → scheda "Generale"
  6. Scorri fino a "Le tue app" → clicca </> (Web) → dai un nome → Registra app
  7. Copia il blocco firebaseConfig = { ... } e incollalo qui sotto
  8. Condividi questa stessa configurazione con tutti i familiari: tutti useranno lo stesso database
